Impinj R2000 protocol. 
=========================
 
    pyImpinj is a protocol library for R2000 reader modules.


Installation
------------

* Compile and install the library::

    pip3 install pyImpinj

Usage
-----

  **In Python 3::**
    from pyImpinj import pyImpinj

Example
-------

    from pyImpinj import pyImpinj
